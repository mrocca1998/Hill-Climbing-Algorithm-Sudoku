To run my project code:
1) Download the zip file submission
2) Unzip the file in the directory of your choosing
3) navigate to that directory
4) Place your input file(s) in the same directory as the .java files

5)
To compile the HillClimber code enter the command "javac HillClimber.java"

6) 
To run the HillClimber enter the command "java HillClimber filename"
With filename being the name of the input data file
